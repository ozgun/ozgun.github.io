svn komutları:

yeni repository tanımlamak için: 

# svnadmin create /home/ozgun/Repositories/devel_repo

Bir projei devel_repo'ya yüklemek için:(Dosyalar trunk içinde olacak) 

# mkdir ruby{trunk,branches,tags}
# touch ruby/trunk/file1.rb
# svn import ruby file:///home/ozgun/Repositories/devel_repo

ruby projesini devel_repo'dan çalışma dizinine indirmek için 

# cd /home/ozgun/Devel
# svn checkout file:///home/ozgun/Repositories/devel_repo/trunk ruby

Yeni bir dosya eklemek için 

# cd /home/ozgun/Devel/ruby/workspace
# touch file2.rb
# svn add file2.rb
# svn add file3.rb
# svn commit -m "file2.rb ve file3.rb eklendi"

Dosya silmek 

# svn delete file3.rb
# svn commit -m "file3.rb silindi"

Dosya'nın adını değiştirmek. 

# svn move file2.rb file2x.rb
# svn commit -m "file2.rb'nin adi file2x.rb oldu"

Dosya kopyalamak 

# svn copy file1.rb file4.rb
# svn commit -m "file1.rb dosyas&#305; file4.rb'ye kopyaland&#305;."

Bir dosyanın logunu görmek. 

# svn log dosya.txt

Bir dosyanın belli bir revizyonunun görmek. 

# svn cat -r 1 dosya.txt

Binary dosyanın belli bir revizyonun görmek. 

# svn cat -r 1 dosya.png > /tmp/dosya.png

Aktif Internet bağlantılarını göster 

# lsof -Pni

# ecasound -i:/dev/dsp -o:deneme.wav
