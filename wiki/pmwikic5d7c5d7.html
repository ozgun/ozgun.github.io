Perl: Cpan

cpan ile Perl Modullerinin Kurulumu

Modül kurulumu ve örnekler.

cpan> install ModulAdi
cpan> install Wx
cpan> install Bundle::Cpan

cpan shell'ini çalıştırmadan modul kurmak.(2 yöntem var)

# cpan -i Netscape::Booksmarks Business::ISBN
# perl -MCPAN -e install Acme::Bleach

Zorba - Zorla modül kurulumu

cpan> force install ZorlaKurulacakModul

Modül Arama:

cpan> i /aranacak modul/

cpan konfigurasyonunu en baştan yapmak için:

cpan> o conf init

Tüm cpan konfigurasyonunu görmek için:

cpan> o conf

Yansı listesine yeni bir adres eklemek için. Ekledikten sonra değişiklikleri kaydederiz.

cpan> o conf urllist push ftp://ftp-mirror.internap.com/pub/CPAN/
cpan> o conf commit

Modullerin listesini bir dosyaya yazdırmak:

cpan> autobundle

Debugging: Örneğin bir modülü kurmaya çalışırken sorun çıktı,sorunu tespit etmek için aşağıdaki yöntemler izlenebilir.

cpan>install Wx
cpan>look Wx (Bu komut ile bir subshell açilir.)
mor:~/.cpan/build/Wx-0.26# make test TEST_VERBOSE=1 TEST_FILE=t/04-misc.t | tee test.log
mor:~/.cpan/build/Wx-0.26# exit

Tüm modüllerin güncellenmesi(Büzük ister biraz bu komut, denemedim ben)

# perl -MCPAN -e 'CPAN::Shell->install(CPAN::Shell->r)'
