Debian'da Network Ayarlari

Statik IP için /etc/network/interfaces dosyasında yapılacak ayarlar:

# /etc/network/interfaces -- configuration file for ifup(8), ifdown(8)
# The loopback interface
# automatically added when upgrading
auto lo
iface lo inet loopback
# The first network card - this entry was created during the
# Debian installation automatically added when upgrading
auto eth0
iface eth0 inet static
 address 192.168.1.2
 netmask 255.255.255.0
 network 192.168.1.0
 broadcast 192.168.1.255
 gateway 192.168.1.1

Dinamik(dhcp) IP için /etc/network/interfaces dosyasında yapılacak ayarlar:

# /etc/network/interfaces -- configuration file for ifup(8), ifdown(8)
# The loopback interface
# automatically added when upgrading
auto lo
iface lo inet loopback
# The first network card - this entry was created during the
# Debian installation automatically added when upgrading
auto eth0
iface eth0 inet dhcp

Ayarları yaptıktan sonra:

# ifup eth0
