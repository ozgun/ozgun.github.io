Ruby: Mysql

Ruby MySQL?

"mysql" driver'ının kurulumu:

# gem install mysql

Veritabanına bağlanılması:

require 'mysql'
#my = Mysql.new(hostname, username, password, databasename)
con = Mysql.new('localhost', 'testuser', 'testpass', 'testdb')

Veritabanı sorgusu yapılması:

rs = con.query('select * from testtable')

Veritananından getirilen kayıtların yazdırılması:

rs.each_hash {|h| puts h['column_name']}

Veritabanı bağlantısının sonlandırılması.

con.close
