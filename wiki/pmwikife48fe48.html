Linux: Ding

Kurulumu

sudo apt-get install ding

İngilizce kelimelerin anlamına internetten bakmak için "dict"i de kurarız.

sudo apt-get install dict

Konfigurasyon

Programın öntanımlı arayüzü Almanca, o yüzden ilk çalıştırdığımzda dilini İngilizce'ye çevirmek benim gibi Almanca özürlü insanlar için olmazsa olmaz.. Bunun için soldan 2. menüden "Allgemein"i seçeriz. Açılan pencerede "Sprache" bölümünde dili "English" olarak ayarlarız. "Ubernehmen" e tıklayarak onaylayıp bu pencereyi kapatırız. Programı kapatıp tekrar açtığımızda İngilizce arayüzü ile bizleri karşılayacaktır.

Kelimelerin yazılışlarının kontrol edilmesi(İngilizce)

Kelime arama kutusuna yazılışını kontrol edeceğimiz kelimeyi yazarız. Yandaki menüden de "Spell check"i seçip, "Search" düğmesine tıklarız.
