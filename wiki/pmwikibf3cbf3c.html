Smartmontools

Smartmontools

Kurulum

# apt-get install smartmontools

Konfigurasyon

# vim /etc/default/smartmontools

enable_smart="/dev/hda /dev/hdb"
start_smartd=yes
smartd_opts="--interval=1800"

Disklerde smart Desteğinin Kontrol Edilmesi 

# smartctl -i /dev/hda

Yukarıdaki komutun çıktısı(Diske göre değişir) 

=== START OF INFORMATION SECTION ===
Device Model:     Maxtor 6E040L0
Serial Number:    E10QZ31N
Firmware Version: NAR61EA0
Device is:        In smartctl database [for details use: -P show]
ATA Version is:   7
ATA Standard is:  ATA/ATAPI-7 T13 1532D revision 0
Local Time is:    Wed Feb  8 16:50:19 2006 EET
SMART support is: Available - device has SMART capability.
SMART support is: Enabled

smart Desteğinin Verilemesi 

# smartctl -s /dev/hda
# smartctl -s /dev/hdb

Disklerin Kontrol Edilmesi 

# smartctl -Hc /dev/hda
# smartctl -Hc /dev/hdb

Disklerin Hangi Testleri Desteklediğinin Görülmesi 

# smartctl -c /dev/hda
# smartctl -c /dev/hdb

Self-Testlerin Uygulanması 

# smartctl -t short /dev/hda
# smartctl -t long /dev/hda

Başlatılan Bir Testin İptal Edilmesi 

# smartctl -X

Self-Test Sonuçlarının Görülmesi 

# smartctl -l selftest /dev/hda
# smartctl -l error /dev/hda

Konfigurasyon dosyası 

/etc/smartd.conf
